WPMI Novembre | Utilizing WP Rest API with React Native to login e query data

Hosted By
Giovanni Atreides I. and 3 others

Details

Ciao!
L'appuntamento di novembre è per mercoledì 13 alle 19 con un altro interessante talk in presenza, sempre al Politecnico di Milano!

Questo mese lo speaker sarà Francisco Matelli, con il talk Utilizing WP Rest API with React Native to login e query data
Parleremo della API REST di WordPress e di come possa essere integrata con React Native e Redux per lo sviluppo di applicazioni mobili e web. Esploreremo concetti come il lock-in tecnologico e il design incrementale per capire cosa significhino questi concetti in un contesto in cui WordPress domina oltre il 40% del mercato web.

---------
Stuzzichini e bevande saranno offerti dal nostro sponsor VivaHosting!
---------
Stiamo conducendo, per il secondo anno consecutivo, uno studio sulla percezione della “Diversity, Equity, Inclusion and Belonging” (abbreviato DEIB, in italiano: Diversità, Equità, Inclusione e Appartenenza) all'interno della community italiana di WordPress per ottenere una panoramica più completa delle opinioni delle persone all'interno di questa comunità.
Compilate il SONDAGGIO (vi serviranno solo pochissimi minuti)

In breve, ecco come sarà organizzata la serata:
19:00 Benvenuto + networking
19:30 Presentazione (di circa 30 minuti & Q&A)
20:40 Networking
21:00 Fine; Chi vuole può continuare il networking in un bar o pizzeria vicino al luogo dell’evento. All'inizio dell'incontro faremo il conteggio per prenotare: per non creare disguidi conferma la presenza solo se sei sicuro di venire.

Networking significa che ci sarà tempo per chiacchierare con gli altri partecipanti e fare amicizia. Potresti trovare una persona interessante, un futuro cliente, un futuro fornitore o un nuovo partner di lavoro, chi può dirlo?

Non riesci a venire, o devi dare la tua disdetta? Nessun rancore, sarà per la prossima. Ti chiediamo solo di farci sapere 24 ore prima così - liberiamo il tuo posto per altre persone.

Speaker

Francisco Matelli

Francisco Matelli, sviluppatore software dal 2005 e laureato magistrale in Amministrazione con specializzazione in innovazione (2016). Nel corso della mia carriera, ho partecipato a numerosi progetti tecnologici e attualmente mi occupo di sviluppo web e mobile, con particolare interesse per React, Web3 e Intelligenza Artificiale. Ho iniziato la mia carriera in Brasile e oggi vivo nel nord Italia, dove cerco opportunità di crescita professionale e collaborazioni con aziende europee.
